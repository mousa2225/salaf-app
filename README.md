# 📒 دفتر السلف

نظام محاسبي لإدارة سلف الموظفين، مع كشوف حسابات تفصيلية، استيراد وتصدير إكسل، وتخزين سحابي آمن.

## ✨ المميزات

- 👥 إدارة الموظفين (يدوي أو رفع إكسل)
- 💰 تسجيل السلف الشهرية
- ➖ تسجيل الخصومات بأنواع متعددة (من الراتب، من الإجازة، إلخ)
- 📊 كشف حساب تفصيلي لكل موظف
- 📥 استيراد وتصدير إكسل
- 🔒 تسجيل دخول آمن
- ☁️ تخزين سحابي - شارك بياناتك بين أجهزتك

## 🛠️ التقنيات

- React + Vite
- Firebase (Auth + Firestore)
- Tailwind CSS
- SheetJS (xlsx)
